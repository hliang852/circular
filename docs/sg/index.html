Singapore vs Hong Kong — key differences
Daily buyback filings — SGX Appendix 8D requires one filing per trading day, richer than HKEX's monthly returns. Roll-up to monthly view is done client-side.
Single filing pipe — every SGX announcement (DI, buybacks, results, placings) flows through SGXNET's one API, not two separate portals like HKEX.
REITs & Business Trusts — units instead of shares. All SG data is labelled accordingly based on SGX security type classification.
No Connect flows — Singapore has no mainland-China cross-border flow equivalent. Capital Market short-selling data sourced from SGX daily CSV.
Currency — all figures in SGD. Cross-market comparisons use the root-level currency field per file.
